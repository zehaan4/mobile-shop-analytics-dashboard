# Mobile Shop Analytics Dashboard

An interactive Excel-based business intelligence dashboard designed for mobile phone retailers.

## Features

* Sales Analytics
* Revenue Tracking
* Profit Analysis
* Inventory Monitoring
* Low Stock Alerts
* Brand Performance Analysis
* Payment Mode Insights
* Salesperson Performance Tracking
* Interactive Dashboard

## Tools Used

* Microsoft Excel
* Pivot Tables
* SUMIFS
* COUNTIFS
* Conditional Formatting
* Dynamic Excel Tables

## Business Benefits

* Track daily and monthly sales
* Monitor inventory levels
* Identify top-selling brands and models
* Improve stock planning
* Analyze customer purchasing patterns

## Dataset

1000 realistic mobile shop transactions generated for demonstration purposes.
